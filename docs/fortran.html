Modern Fortran Features

Fortran remains dominant in HPC for scientific computing due to:

  • Array operations and vectorization
  • Mature optimization in compilers
  • Built-in support for complex numbers
  • Efficient memory layout (column-major)

Basic Syntax

program example
  implicit none
  integer :: i, n = 100
  real(8), dimension(n) :: x, y, z
  
  ! Array operations
  x = [(i*1.0, i=1,n)]
  y = 2.0 * x
  z = x + y
  
  ! Matrix multiplication
  real(8), dimension(n,n) :: A, B, C
  C = matmul(A, B)
end program

Module Example

module physics_constants
  implicit none
  real(8), parameter :: c = 2.998e8  ! speed of light
  real(8), parameter :: hbar = 1.055e-34
contains
  function energy(mass) result(E)
    real(8), intent(in) :: mass
    real(8) :: E
    E = mass * c**2
  end function
end module
